# 용어 사전 (Glossary)

> 다우밸브 전사 공통 정의. 부서·사람에 따라 달리 쓰이던 용어를 통일합니다.

---

## 핵심 수치 용어

| 용어 | 정의 | 비고 |
|------|------|------|
| **매출** | 월 출고 기준 세금계산서 발행액 | 수주액·입금액과 구분 |
| **수주** | 고객으로부터 PO(발주서) 또는 확정 발주를 받은 금액 | 매출 확정 전 단계 |
| **출고** | 창고에서 고객에게 물품이 나간 시점 | 매출 인식 기준 |
| **입금** | 고객으로부터 대금이 실제 수령된 시점 | 현금흐름 기준 |
| **마진** | (출고금액 − 원가) / 출고금액 × 100 (%) | |
| **원가** | 제품 매입원가 + 부대비용 | |

## 거래 관계 용어

| 용어 | 정의 |
|------|------|
| **고객사** | 다우밸브에 발주하는 법인 또는 개인. `companies/customers/` 관리 |
| **공급사** | 다우밸브가 매입하는 법인. 목록 → `companies/suppliers/_index.md` |
| **대리인** | 해외 현지에서 다우밸브를 대행하는 파트너. 별도 대리인 계약 체결 |
| **발주** | 다우밸브가 공급사에게 물품을 주문하는 행위. `operations/발주/` 추적 |

---

## 영업 프로세스 용어

| 용어 | 정의 |
|------|------|
| **견적** | 고객 요청에 따라 단가·수량·납기를 정리한 문서 |
| **PO** | Purchase Order. 고객 발주서 |
| **납기** | 고객이 요청한 물품 수령 희망일 |
| **D/A** | 국외 거래에서 서류 인도 후 대금 결제 |
| **L/C** | Letter of Credit. 신용장 |
| **바이어** | 해외영업팀에서 사용하는 해외 고객사 호칭 |
| **AVL** | Approved Vendor List. 승인된 공급사 목록 |
| **MRO** | Maintenance, Repair, Operations. 유지보수용 소모성 자재 |
