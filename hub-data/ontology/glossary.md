# 용어 사전

> 가상밸브 주식회사 업무 용어 정의 (테스트 더미 데이터)

---

## 영업 용어

| 용어 | 정의 |
|------|------|
| 수주 | 고객으로부터 발주서(PO)를 받아 판매가 확정된 상태 |
| 견적 | 고객 요청에 따라 가격·납기·사양을 제시한 문서 |
| 미결업무 | 완료되지 않고 진행 중인 업무 항목 |
| 파이프라인 | 현재 진행 중인 영업 기회의 전체 목록 |
| 납기 | 제품을 고객에게 납품하기로 약속한 날짜 |
| 출고 | 창고에서 제품을 꺼내 고객에게 발송하는 행위 |

---

## 제품 용어

| 용어 | 정의 |
|------|------|
| 볼밸브 | 볼 형태의 마개로 유체 흐름을 제어하는 밸브 |
| 게이트밸브 | 게이트(판)를 올리고 내려 유체를 차단하는 밸브 |
| 체크밸브 | 유체가 한 방향으로만 흐르도록 하는 역지 밸브 |
| DN | Diameter Nominal, 배관 공칭 직경 (mm 단위) |
| PN | Pressure Nominal, 공칭 압력 등급 |

---

## 채널 용어

| 용어 | 정의 |
|------|------|
| 반도체 채널 | 반도체 제조 공정용 고순도 밸브 납품 경로 |
| 2차전지 채널 | 배터리 제조 설비용 밸브 납품 경로 |
| 일반산업 | 화학·식품·건설 등 범용 산업 고객 |
| 대리점 | 특정 지역 또는 분야의 독점/비독점 판매 파트너 |
