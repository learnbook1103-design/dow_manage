## 데이터 현황 — 필독

**ERP 미연동 상태입니다.**
이 허브의 모든 데이터는 Google 공유드라이브에서 특정 시점에 수동으로 가져온 스냅샷입니다.
실시간 데이터가 아니며, 원본이 Drive에서 변경되어도 이 허브는 자동으로 갱신되지 않습니다.

| 데이터 | 스냅샷 기준일 | 원본 위치 |
|--------|-------------|---------|
| 팀별 미결업무 (sales/pipeline.md) | 2026-04-07 | Drive `미결업무(팀별작성).xlsx` |
| 공급사 목록 (companies/suppliers/_index.md) | 2026-04-09 | Drive `◎ 가격표 (매입-공급사별)/` |
| 조직·인사 (team/members.md) | 2026-04-09 | Drive `근태관리_DB` employees 시트 |
| 클레임 DB (sales/analysis/claims.md) | 2026-04-09 | Drive `클레임 DB` Google Sheet |
| 해외 미팅일지 (companies/customers/HPRAY) | 2026-04-09 | Drive `★Meeting Minuets (2026).xlsx` |

**AI 작업 시 주의:**
- 위 데이터를 "현재 상태"가 아닌 "해당 시점 스냅샷"으로 취급할 것
- 수치(금액, 진행율 등)를 단정하지 말고 출처와 기준일을 함께 언급할 것
