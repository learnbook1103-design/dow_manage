# 조직 구조 (Org Chart)

> 출처: 미결업무(팀별작성).xlsx + 공유 드라이브 개인 폴더 분석  
> 최종 업데이트: 2026-04-09

---

## 전체 구조

```
대표이사 (박의균)
│
├── 해외영업팀        → 수출 바이어 영업 (이대희 차장)
├── 국내외 관리영업팀  → 국내 고객사 영업 + 관리 업무 (김혜원 부장)
├── 2차전지 영업팀    → 2차전지 채널 전담 (이재근 차장)
├── 반도체영업팀      → 반도체 채널 전담 (정미영 전무)
├── 밸브파크팀        → 온라인 판매 채널 (곽정원 과장)
├── 엔지니어링팀      → 도면·데이터시트·기술 문서 (서현석 차장)
├── 생산기술팀        → 생산 관리·안전 업무 (안효모 차장)
└── 관리부           → 경리·회계·경영지원 (김현욱 선임)
```

---

## 팀별 주요 역할

| 팀 | 핵심 업무 | 주요 데이터 |
|----|---------|-----------|
| 해외영업팀 | 수출 바이어 관리, 해외 전시회, 수출 매뉴얼 | Meeting Minutes, 바이어별 견적·수주 |
| 국내외 관리영업팀 | 국내 영업, 협력사 계약, 가격체계 관리 | 발주서, 거래명세서 |
| 2차전지 영업팀 | 2차전지 고객사 영업, A/S 관리 | 클레임 DB, 고객 솔루션 사례 |
| 반도체영업팀 | 반도체 고객사 영업, 재고 보고 | 조달청 판매 조사 |
| 밸브파크팀 | 스마트스토어·자사몰 운영, 뉴스레터 | 판매가 비교표, 매입매출 리스트 |
| 엔지니어링팀 | 도면 프로그래밍, Data Sheet, 절차서 | 도면 파일, Data Sheet 양식 |
| 생산기술팀 | 생산 안전, 구역 표시 | 생산 절차서 |
| 관리부 | 경리·회계, ERP, 경영보고 | 더존, 매출채권보험, 연차관리 |

---

## 영업 파이프라인 관할

```
국내 고객사   →  국내외 관리영업팀 + 반도체영업팀 + 2차전지 영업팀
해외 바이어   →  해외영업팀 (단독 관할)
온라인 채널   →  밸브파크팀 (단독 관할)
```
