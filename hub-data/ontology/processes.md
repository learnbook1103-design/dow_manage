# 업무 흐름 (Processes)

---

## 영업 프로세스

```
리드 발굴
    ↓
견적 요청 수신
    ↓
견적서 작성·발송
    ↓
고객 검토 / 협의 (가격·납기·사양)
    ↓
수주 확정 (PO 수령)
    ↓
생산 발주 / 재고 확인
    ↓
납기 조율
    ↓
출고 처리
    ↓
세금계산서 발행 → 매출 인식
    ↓
대금 입금 → 정산 완료
```

---

## 국내영업 vs 국외영업 차이점

| 구분 | 국내영업부 | 해외영업부 |
|------|-----------|-----------|
| 결제 방식 | 세금계산서 + 계좌이체 | L/C, T/T, D/A 등 |
| 견적 단위 | 원화 (KRW) | 달러/유로 등 외화 |
| 납기 기준 | 고객사 도착일 | 선적일 (ETD) 또는 도착일 (ETA) |
| 수출 서류 | 불필요 | 인보이스, 패킹리스트, B/L 등 |

---

## 파이프라인 갱신 기준

| 이벤트 | 파이프라인 변경 내용 |
|--------|---------------------|
| 견적 요청 수신 | 단계 → `견적요청`, 예상 금액 입력 |
| 견적서 발송 | 단계 → `견적발송`, 발송일 기록 |
| 고객 협의 시작 | 단계 → `협의중`, 주요 쟁점 메모 |
| PO 수령 | 단계 → `수주확정`, 수주금액 확정 |
| 생산/납기 진행 | 단계 → `생산/납기`, 납기일 기록 |
| 출고 완료 | 단계 → `출고완료`, 출고금액 기록 |
| 수주 실패 | 단계 → `종료`, 실패 사유 기록 |
