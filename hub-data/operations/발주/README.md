# 발주 현황

> 다우밸브가 공급사에게 발주한 내역을 추적합니다.

---

## 개요

- **담당**: 각 영업팀 + 관리부 연계
- **원본 위치**: 공유 드라이브 → 발주서 폴더
- **현재 상태**: ERP 개발 완료 후 자동 연동 예정 (D-003 완료 시)

---

## 발주 프로세스

```
고객 수주 확정 (PO 수령)
    ↓
재고 확인 (재고/ 참조)
    ↓
공급사 발주 (거래처 → companies/suppliers/_index.md 참조)
    ↓
납기 확인
    ↓
입고 확인 (입출고/ 기록)
    ↓
출고 처리
```

---

> 발주 이력은 이 폴더에 날짜별 파일로 저장합니다.
> 예: `2026-04-27-발주내역.md`
